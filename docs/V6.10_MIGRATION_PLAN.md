### 3. **Circuit Breaker — Stop After 3 Losses**

**Pine v6.10 Line 1549-1552:**
```pine
var int consecutiveLosses = 0
var int circuitBreakerBar = -999
int circuitBreakerCooldown = 10 + math.max(0, consecutiveLosses - 3) * 10
bool circuitBreakerOn = consecutiveLosses >= 3 and (bar_index - circuitBreakerBar) < circuitBreakerCooldown
```

**Logic:**
- After 3 consecutive SL hits: pause 10 bars
- 4th loss: +10 bars (20 total)
- 5th loss: +10 bars (30 total)
- Escalating cooldown prevents repeated-loss chains

**Current Python:** No circuit breaker

**Action:** Implement in `scanner.py` trade tracking — log consecutive losses, block new signals when ≥3.

1. **§8 Indicators — Add P5 Candle Body**
```python
def calc_candle_body_score(df):
    """P5: Candle Body Confirmation (+2 score)
    Bull: close > open, body > 50% of bar range
    Bear: close < open, body > 50% of bar range
    """
    bar_range = df['high'] - df['low']
    body = abs(df['close'] - df['open'])
    body_pct = body / bar_range
    
    p5_bull = (df['close'] > df['open']) & (body_pct > 0.5)
    p5_bear = (df['close'] < df['open']) & (body_pct > 0.5)
    
    return p5_bull.astype(int) * 2, p5_bear.astype(int) * 2
```

```python
# In active_trades.json schema:
{
    "consecutive_losses": 0,
    "circuit_breaker_until_bar": null,
    "last_loss_bar": null
}

def check_circuit_breaker(state):
    if state['consecutive_losses'] < 3:
        return False
    
    bars_since_loss = current_bar - state.get('last_loss_bar', 0)
    escalation = max(0, state['consecutive_losses'] - 3) * 10
    cooldown = 10 + escalation
    
    return bars_since_loss < cooldown

def on_sl_hit(state):
    state['consecutive_losses'] += 1
    state['last_loss_bar'] = current_bar
    if state['consecutive_losses'] >= 3:
        state['circuit_breaker_until_bar'] = current_bar + 10

def on_tp_hit(state):
    state['consecutive_losses'] = 0  # reset on any win
```
