## Part 1: v6.10 Deep Review

### Architecture (5-pillar, 12pt max)

| Pillar | Weight | Source Logic | Quality |
|--------|--------|--------------|---------|
| P1 HTF Trend | 3 | 4H EMA50 vs EMA200 | ✅ Solid — captures macro |
| P2 Momentum | 2 | MACD bull OR (RSI>50 AND rising) | 🟡 OR-logic dilutes — should be AND |
| P3 Volume | 3 | high vol + (CVD or vol>3-bars-ago) | 🟡 OR-logic again, CVD optional |
| P4 Regime | 2 | TRENDING or VOLATILE | ✅ Hard gate also — good |
| P5 Candle Body | 2 | body > 50% of bar range | ✅ Strong v6.10 add |

### Gate stack (sequential filtering)

```
Raw trigger (pullback / base / flip)
    ↓
ADX gate (≥20 or bypass)
    ↓
Score + Grade (12pt threshold)
    ↓
Pillar purity (≥3 of 5)
    ↓
Counter-trend block (severity-weighted)
    ↓
Quality gate (regime + slope + ST + VWAP + POC + EMA ext + chop + gap)
    ↓
Time filter + leg state + barsSinceSig + chop dist + circuit breaker + SQS
    ↓
CONFIRMED entry
```

### Strengths
1. **Layered gates** — each filter independent, easy to audit via Gate Funnel table
2. **TF-aware tuning** — 15m/30m/1H+ branches calibrated per timeframe
3. **Circuit breaker** — 3-loss escalating cooldown prevents tilt streaks
4. **Structure-based SL** — uses recent swing low/high + ATR floor/cap
5. **Dynamic regime TP/SL** — VOLATILE wider, QUIET tighter
6. **Smart re-entry** — re-arms after TP1 hit + pullback
7. **POC ATR-based gate** — scales with volatility (not fixed %)
8. **Pillar purity** — prevents 12pts from 2 strong pillars + 3 zeros
9. **SQS 5-factor weighted** — Regime 28% + Confluence 25% + MTF 20% + Vol 15% + Volatility 10%

### Weaknesses for PF/trades

#### W1: P2/P3 OR-logic dilutes signal
```pine
// v6.10 line 1301:
bool p2Bull = macdBull or (rsiVal > 50 and rsiVal > rsiVal[2])
```
MACD bull alone gives 2pts even if RSI weak. Drops entry quality.

**Fix v7.0:** AND-then-half-credit
- Both true → 2pts
- Either true → 1pt
- Neither → 0pts

Same for P3 (volume + CVD).

#### W2: No mean-reversion path
v6.10 only fires on pullback-resumption / base-resume / flip. Misses:
- VWAP -2SD reclaim
- Oversold bounce off EMA200
- Liquidity sweep + reclaim (rawSweep detected but gates kill entry)

**Fix v7.0:** Add mean-reversion entry trigger. Separate scoring/SL profile.

#### W3: SL too wide on VOLATILE
v6.10 maxSLDistVol = 5.0 ATR. IREN/OKLO swing 2.5-3.5 ATR. Wide SL = bigger losses → drops PF.

**Fix v7.0:** Stage SL — initial 5.0 ATR cap, tighten to 3.0 ATR after 0.5R move (price has shown direction).

#### W4: Single trail multiplier
v6.10 trailAtrMult = 1.8 fixed. Works for trending, gives back too much in choppy post-TP3.

**Fix v7.0:** Tighten trail multiplier per TP stage:
- After early BE: 1.8 (current)
- After TP1: 1.5
- After TP2: 1.0
- After TP3: 0.6 (lock most gains)

#### W5: No partial exit logic in indicator
Pine v6.10 backtest assumes 40/35/25 split but the SIGNAL fires single TP. User has to manage manually.

**Fix v7.0:** Document partial exit recommendation in signal alert (40% at TP1 / 35% TP2 / 25% trail).

#### W6: Choppiness Index 1H+ only
v6.10 line 704: chop gate only on 1H+. 15m/30m pullback-resumption fires in consolidation chop where it shouldn't.

**Fix v7.0:** Add direction-aware chop gate on 30m too (line 704 condition relaxed). 15m left out for trade count.

#### W7: No time-of-day quality scoring
US market 10-11am ET and 2-3pm ET have best directional follow-through. 11:30-1:30 lunch chop. v6.10 has no time-quality weight.

**Fix v7.0:** Time-of-day quality multiplier on SQS (+5 during 10-11/2-3, -10 during 11:30-13:30 lunch).

#### W8: No RS vs SPY/QQQ in confluence
v6.10 has MTF gate but no relative strength vs index. Long signal on weak stock vs strong SPY = lower win rate.

**Fix v7.0:** Add RS confluence bonus (+1pt if symbol outperforming SPY 5d, -1pt if underperforming during long).

#### W9: Re-entry too restrictive
useReEntry default OFF + min score 6 + reEntryQualGate too tight. Re-entries rare even when configured.

**Fix v7.0:** Default ON, lower min score (5), simpler quality gate (regime + ST only).

#### W10: Big mover alert independent but no follow-up gate
Big mover detects ±5% day move but doesn't pass info to signal engine. Stock up 8% on news may keep going.

**Fix v7.0:** Big mover detection adds momentum tag, biases gates toward continuation (lowers EMA ext from 8.0 to 10.0 ATR temporarily on momentum days).

### Python scoring rewrite
```python
def calc_pillar_score(df, htf_bull, regime, rs_pct):
    """v7.0: weighted 5-pillar with AND-logic and RS bonus."""
    last = df.iloc[-2]  # closed bar
    
    # P1 HTF (3pts)
    p1 = 3 if htf_bull else 0
    
    # P2 Momentum (2pts AND, 1pt OR, 0 neither)
    macd_bull = last['macd'] > last['macd_signal']
    rsi_bull  = last['rsi'] > 50 and last['rsi'] > df['rsi'].iloc[-4]  # 2 bars rising
    if macd_bull and rsi_bull:
        p2 = 2
    elif macd_bull or rsi_bull:
        p2 = 1
    else:
        p2 = 0
    
    # P3 Volume (3pts AND, 1pt OR)
    high_vol  = last['volume'] > df['volume'].rolling(20).mean().iloc[-2] * 1.1
    cvd_bull  = last['cvd'] > df['cvd'].iloc[-4]
    if high_vol and cvd_bull:
        p3 = 3
    elif high_vol or cvd_bull:
        p3 = 1
    else:
        p3 = 0
    
    # P4 Regime (2pts)
    p4 = 2 if regime in ('TRENDING', 'VOLATILE') else 0
    
    # P5 Candle body (2pts)
    bar_range = last['high'] - last['low']
    body = abs(last['close'] - last['open'])
    body_pct = body / bar_range if bar_range > 0 else 0
    p5 = 2 if (last['close'] > last['open'] and body_pct > 0.5) else 0
    
    # RS bonus (+1pt if outperforming SPY 5d)
    rs_bonus = 1 if rs_pct > 0 else 0
    
    total = min(12, p1 + p2 + p3 + p4 + p5 + rs_bonus)
    return {
        'total': total,
        'p1': p1, 'p2': p2, 'p3': p3, 'p4': p4, 'p5': p5,
        'rs_bonus': rs_bonus,
        'grade': 'A+' if total >= 12 else 'A' if total >= 9 else 'B' if total >= 6 else 'C',
    }
```
